Даний код це моя реалізація шахматів один на один, без АІ (реалізація із АІ на лінуксі я не знаю чи можлива),
але якшо я буду мати натхнення я перекомпілюю цей код на Віндовсі і спробую додати АІ.
Для того щоб бінарник програми запустився на лінуксі я зроблю короткий алгоритм дій:

1. Розархівувати файл.

2. В терміналі прописати команду "sudo apt-get install libsfml-dev".

3. В терміналі прописати команду "./chess".

4. Щоб закрити програму в будь-який момент нажміть Esc.

Усе, приємної гри!
